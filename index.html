HTML

  • The head element contains information about the webpage, like meta tags.
  • The body element represents the visible content shown to the user.
  • The header element contains visible conent shown at the very top of the page, like a directory.
  • The footer element contains visible content shown at the very bottom of the page, like copyright.
  • Some common tags are:
    1. The h1 - h6 tags help separate sections of content (through decreasing heading sizes) and are important to formatting for reader assistance programs.
    2. The ul and ol tags add unordered and ordered lists respectively and li tags add separte items to those lists.
    3. The !doctype html tag at the very top of the html document features no closing tag and lets the webpage know that the document being rendered is in HTML.
    4. The title tag changes the text shown in website listings and the title shown on the browser tab.
    5. The p tags represent a paragraph element
  • Child elements are when three elements are nested inside a body element and nested elements are when an element is nested inside another element.

CSS

  • CSS is also known as Cascading Style Sheets, used to change the apperance and style of HTML documents.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A . preceding a class makes it a class selector, it will style any HTML element it is assigned to.
  • You can assign colors via names or hexadecimal codes.

Git

  • Git is used for version control.
  • Common Commands:
    1. cd (directory name): To change directory
    2. cd .. : To go back two directories
    3. git add: To add file to add files to staging
    4. git branch: List of all branches in current repository
    5. git checkout (branch): To swap current branch
    6. git status: checks what branch we are currently on
    7. git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Javascript is what gives websites functionality.
  • A variable stores data in code: var 'container name' = "data";
  • A string is text, which will be contained in quotation marks.
  • Numbers and boolean values like true, don't need to be contained in quotation marks.
  • To display something in console, console.log(text);
  • To link the javascript file to the html file, you put the link under the footer section above the closing tag for body in the html section.
  • If and else statements check for true or false conditions and only run code if found truthy.
  • An array is a single variable holding a big group of data: var 'container name' = [data,data,data];
  • If we want to access a specific piece of data from an array we can use the variable and number starting at zero: console.log(variable[0]);
  • A for loop is an easy way to execute code on all the data in an array.
  • The first part of a for loop has the starting point like var x = 0, second is the condition x less than shapes.length;, third is the repeat like x++.
  • A function is a piece of code telling the computer how to execute a basic task.
  • To write a function: function 'name'() { // code block}
  • To call a function you write the name of the function followed by ()
  • To generate a random number you can use: Math.random()
